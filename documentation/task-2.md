# Dashboard / Landing Page Schüler

## Description:
- Als eingeloggter Schüler möchte ich nach dem Login eine übersichtliche Startseite (Dashboard) sehen, damit ich schnell einen Überblick über meine Möglichkeiten, aktuellen Rückmeldungen und wichtigen Informationen erhalte.

## Acceptance Criteria

- Nach dem Login wird der Schüler automatisch auf das Dashboard weitergeleitet

- Das Dashboard enthält:

    - eine Übersicht meiner zuletzt eingereichten Feedbacks (z. B. letzte 3)

    - den aktuellen Status dieser Feedbacks (neu / gesehen / beantwortet)

    - einen direkten Zugang zum Feedback-Formular

    - einen direkten Zugang zum Petitionsbereich

- Neue oder unbeantwortete Rückmeldungen sind visuell hervorgehoben

- Das Dashboard ist übersichtlich und auch auf kleineren Bildschirmen gut nutzbar
